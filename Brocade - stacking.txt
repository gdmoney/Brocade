============================
General:

stack enable				                   // individually on each switch
stack secure-setup
stack suggested-id ...			          // If bootup ID not available, use this
stack switch-over			               // standby becomes active
show stack

============================

Replacing a stack unit:

- load the same code version on the new stack member
- connect new unit to the stack before powering it on
- enter: stack enable on the new unit
- enter: stack secure-setup to renumber the new unit (optional)
- configure the new unit's interfaces
